## 📖 Why I Built This

While practicing DSA, I realized I wasn't struggling with algorithms as much as I was struggling with **consistency**.

Some days I'd spend hours on one problem.

Other days I'd look at hints too early.

Even after solving a question, I'd often skip:

- Taking notes
- Comparing with the optimal solution
- Analyzing my own code

Eventually, I created a workflow that worked.

The problem?

I didn't always follow it.

So I built **Deadline Clock**—a timer that guides you through every stage of solving a problem instead of giving you one unlimited countdown.

The goal isn't to solve problems faster.

It's to make sure you don't skip the parts that actually make you better.

# ✨ Features

## 🧠 Workflow-Based Learning

Instead of a single timer, every problem follows a structured workflow.

- 👀 Read & Restate
- 🧠 Solve (No Hints)
- 💡 Hints
- ⚡ Compare with Optimal
- 🔍 Analyze Your Code
- 📝 Notes

---

## ⏱ Difficulty-Based Timers

Different workflows for:

- Easy
- Medium
- Hard

Each with customizable durations.

---

## ⚙ Fully Customizable

Build your own workflow.

- Add custom steps
- Remove unnecessary ones
- Rearrange stages
- Customize every timer

Perfect for DSA, competitive programming, revision sessions, or even studying.

---

## ☁ Firebase Cloud Sync

Create an account and your progress follows you everywhere.

- Theme preferences
- Timer settings
- Custom workflows
- Session data

Automatically synced across devices.

---

## 📊 Question History

Every completed problem is recorded.

Track:

- Time spent
- Workflow followed
- Completion history

---

## 🪟 Mini Mode & Picture-in-Picture

Keep the timer floating while solving problems.

Perfect when switching between:

- LeetCode
- VS Code
- Browser
- Notes

No need to constantly switch tabs.

---

## 🎨 Personalization

Choose from multiple themes.

Make your workspace your own.

---

## ☕

Smart break reminders help prevent burnout during long study sessions.

---

## 💾 Automatic Saving

Everything is automatically saved.

Even if you accidentally close the browser.
